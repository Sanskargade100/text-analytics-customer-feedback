# text-analytics-customer-feedback
MSc Text Analytics Coursework - Task 4: Mining customer feedback

## Task 4 Report Structure

The final report analyses customer support tickets using two analytic axes:

- Axis 1: Text representation, comparing TF-IDF and multilingual embeddings.
- Axis 2: Topic modelling, comparing LDA and BERTopic.

The report connects these axes to the evaluation section through clustering stability metrics, topic coherence, topic size distributions, representative examples, and qualitative interpretability analysis.
